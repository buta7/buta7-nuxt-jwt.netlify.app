# nuxt-jwt-example

## Build Setup

```shell
npx create-nuxt-app nuxt-jwt-example
npm run dev
```

```shell
npm install --save @nuxtjs/axios @nuxtjs/auth @nuxtjs/vuetify
npm install --save dotenv
```

## Mock API server

```
cd express-jwt-mock
make
サーバを起動しました。http://localhost:8080
```
